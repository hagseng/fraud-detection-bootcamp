# fraud-detection-bootcamp
Zero2End Machine Learning Bootcamp bitirme projesi. Dengesiz veri (imbalanced data) üzerinde Sınıflandırma, Regresyon, Ranking ve Gözetimsiz Öğrenme teknikleri kullanılarak geliştirilen, Streamlit ile canlıya alınmış uçtan uca Kredi Kartı Dolandırıcılık Tespit (Fraud Detection) sistemi.
